NodeJS: Criando primeira biblioteca

O que é uma biblioteca?
- Pacote de código utilizado para automatizar ou otimizar uma tarefa, e podemos importar ou exportar.

Node é um conjunto de comandos para rodar código JS fora do navegador.

npm é um gerenciador de pacotes do node. Gerenciadores de pacotes são repositórios de código aberto nos quais devs disponibilizam soluções para o uso da comunidade.
